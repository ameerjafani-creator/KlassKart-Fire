# **App Name**: Klass Kart

## Core Features:

- Firebase Unified Authentication: Secure sign-in utilizing Google and Email/Password with specific role-based access for admins and users.
- Dynamic Catalog Engine: Comprehensive product browsing with advanced sorting, multi-category filtering, and Firestore-powered search.
- Integrated Cart & Wishlist: High-performance persistence of user selections with real-time Firestore synchronization across sessions.
- AI Personal Shopper Tool: A generative tool that recommends curated product lists based on trending items and historical user browsing patterns.
- Cashfree Payment Terminal: Secure transaction gateway supporting UPI, Cards, and Net Banking with server-side validation tools to verify success.
- Admin Operational Suite: Dedicated command center for managing inventory, category hierarchies, and viewing real-time sales analytics.
- Order Lifecycle Tracking: Automated fulfillment monitoring from checkout to delivery including invoice generation and low-stock alerts.

## Style Guidelines:

- The primary brand color is a bold, high-contrast Red (#D11D2C) to evoke urgency and premium energy.
- The interface sits on a high-clarity Off-White background (#FAF9F9) to emphasize product photography.
- The accent color is a Deep Charcoal (#2B2324), providing a sophisticated frame for layouts and primary navigation.
- Pairing 'Poppins' for bold, modern headlines with 'Inter' for clear, highly legible body text.
- Slim, minimalist monochrome line icons used consistently for utility and navigational cues.
- A mobile-first, sticky-navigation architecture with soft-shadowed cards and significant white space for a clean editorial look.
- Subtle transition effects and micro-interactions on button states and image hover states for a polished feel.
